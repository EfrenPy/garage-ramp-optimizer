# Security policy

This is a small offline desktop tool (it does not run a server, does
not collect telemetry and does not handle credentials), so the
attack surface is limited.  That said, please report any vulnerability
you find privately first so we can fix it before disclosure.

## Reporting a vulnerability

Please open a private vulnerability report on GitHub:

  https://github.com/EfrenPy/garage-ramp-optimizer/security/advisories/new

I will respond within a few business days.

For non-security bugs and feature requests, please open a regular
[issue](https://github.com/EfrenPy/garage-ramp-optimizer/issues).

## Supported versions

Only the latest release on the `main` branch is supported.  When a
fix is shipped, please update to the newest published release.
